Tolomuco Real Estate API

An example data source to be used with the Tolomuco framework.

Usage

The API is available at /api/ and can take the following GET arguments

per_page
default: 10 - the number of results per page
page
default: 1 - the page number. First page is 1.
total
default: per_page * 2 - the number of results in total
seed
default: 1337405335 - the random number genorator seed.
